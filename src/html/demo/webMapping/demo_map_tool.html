Upload Data
Files are loaded only into the current map session; they are not uploaded to storage or a server. Supported: JSON, GeoJSON, CSV, TXT, WKT/WTK, and simple DXF point/circle entities. JSON may be flat, GeoJSON, or nested/grouped such as week → vessel → lat/lon. For DXF/WKT/CSV files with projected X/Y coordinates, use Auto-detect or select the correct input CRS before uploading. DWG is proprietary and should be converted to DXF first.

Coordinate Reference System
EPSG mode accepts only an EPSG number and loads the definition automatically. Proj4 mode lets you paste your own projection definition. The basemap remains Web Mercator; only the coordinate readout and pin-editor coordinates are transformed.
