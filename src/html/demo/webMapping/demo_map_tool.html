Upload Data
Files are loaded only into the current map session; they are not uploaded to storage or a server. For loose shapefiles, select .shp, .shx and .dbf together, or use Upload Folder. Optional files: .prj and .cpg. Supported: JSON, GeoJSON, CSV, TXT, WKT/WTK, zipped ESRI Shapefile, loose ESRI Shapefile sidecar groups (.shp + .shx + .dbf + optional .prj/.cpg) selected together or through Upload Folder, and simple DXF point/circle entities. JSON may be flat, GeoJSON, or nested/grouped such as week → vessel → lat/lon. For DXF/WKT/CSV files with projected X/Y coordinates, use Auto-detect or select the correct input CRS before uploading. DWG is proprietary and should be converted to DXF first.
